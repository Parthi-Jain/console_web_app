Project On Console

Visual Studio Code, also commonly referred to as VS Code, is a source-code editor developed by Microsoft for Windows, Linux, macOS and web browsers. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded version control with Git.

A console screen is a computer terminal where a user may input commands and view output such as the results of inputted commands or status messages from the computer.

To open the console screen , you have to right - click your mouse buttton and choose inspect and then choose the console option.
